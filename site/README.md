# AntV Infographic Site

## 快速开始

### 安装

```bash
npm install
```

### 本地开发

启动开发服务器

```bash
# npm
npm run dev
```

然后打开浏览器访问 `http://localhost:3000`

## 项目结构

- `/src/content` - Markdown 文档内容
- `/src/components` - 组件
- `/src/pages` - Next.js 页面路由
- `/public` - 静态资源

## 开发

### 代码检查

执行 Prettier、ESLint 和类型检查：

```bash
npm run check-all
```

### 构建

构建生产版本：

```bash
npm run build
```
